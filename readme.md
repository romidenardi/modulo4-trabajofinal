# Sobre el proyecto

## Objetivo

Desarrollar una API RESTful funcional utilizando Node.js, Express, y MySQL. 

## Requerimientos

* Registro de usuarios: Crear un nuevo usuario con nombre, correo electrónico y contraseña (utilizando hashing de contraseñas). 
* Inicio de sesión: Autenticar de usuarios mediante correo electrónico y contraseña y generar un token JWT. 
* CRUD de usuarios: Permitir la actualización y eliminación de los datos del usuario autenticado. 
* Rutas protegidas: Permitir que los usuarios autenticados puedan acceder a ciertas rutas (por ejemplo, actualizar o eliminar su información). 

La base de datos debe ser MySQL y se debe utilizar Sequelize como ORM para interactuar con la base de datos. 

## Tecnologías y librerías utilizadas

* Node.js.
* Express y Express Validator.
* MySQL2.
* JSON Web Token.
* Sequelize.
* Bcrypt.
* Dotenv.
* Nodemailer.
* EJS.
* Winston.

### Uso de IA

Se utilizó inteligencia artificial generativa para:

* Desarrollar la autenticación de usuarios con JWT.
* Analizar la estructura óptima para el desarrollo del CRUD.
* Agregar validaciones.
* Validar código y corregir errores.
* Analizar y desarrollar funcionalidades extra (envío de email de bienvenida con plantilla EJS y documentación con Winston).
* Estilos CSS para la plantilla EJS.
